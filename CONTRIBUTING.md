 # Contributing to Untitled-JoJo-Gaem


First of all, thanks for taking the time to contribute! This helps to improve the project and be better!


## How Can You Help?


You can contribute in several ways:

- **Programming:** Improving combat logic, optimizing networking, implementing Stand abilities, etc.

- **Bug Reporting:** Opening issues if you find something broken 

- **Assets:** Providing R6 animations,models, or VFX.

- **Suggestions:** Joining the Discord to discuss balance and new features.


## Development Rules


To keep the codebase clean and functional, please follow these standards:


1. **Strict Luau:** Use `--!strict` at the top of every new script.

2. **Mobile First:** Ensure any new input or UI element is compatible with **Android** using `ContextActionService`.


## Pull Request Process


1. **Fork** the repository.

2. Create a new branch for your feature or fix (`git checkout -b feature/AmazingAbility`).

3. Commit your changes with clear, English messages.

4. Submit a **Pull Request** explaining what you changed and why.


## Community & Support


If you want to discuss a major change before working on it, feel free to:

- Open an **Issue**.

- Join our **Discord** (Link in README).

- Tag **Mystic-Red** in your PR.


# Communication


For major changes or architectural proposals, please open an Issue first or discuss it with Agus (Mister Espanto) in the official Discord.
